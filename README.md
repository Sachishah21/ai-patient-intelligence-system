# 🏥 AI Patient Intelligence System

## 📌 Overview

This project is an AI-powered system that analyzes patient clinical documents and generates structured insights using Machine Learning and Large Language Models (LLMs).

It helps improve doctor handovers by summarizing patient history and providing context-aware information.

---

## 🚀 Features

* Extracts text from patient PDF records
* Generates clinical summaries using LLMs
* Uses Retrieval-Augmented Generation (RAG) for context-aware responses
* Supports multiple patient records
* Designed for real-world healthcare assistance

---

## 🛠️ Tech Stack

* Python
* PyPDF2 (PDF processing)
* LLM APIs (Groq)
* Pinecone (Vector Database)
* NLP techniques
